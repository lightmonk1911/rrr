Типовой проект со всей минимально-необходимой структурой. Подойдёт в качестве отправной точки для небольшого проекта.

## Start

```sh
make
```

## Setup
```sh
make build
```

## Run
```sh
make run
```

## Run tests
```sh
make test
```

## Run checkstyle
```sh
make lint
```

## Check update dependencies and plugins
```sh
make update
```

## asdf
